## Основные материалы

Здесь рекомендуется размещать документы и ссылки, необходимые школам для проведения школьного этапа:

- сроки проведения;
- инструкции для ответственного организатора;
- памятки для участников;
- технические требования к аудиториям и рабочим местам;
- ссылки на платформу проведения;
- рекомендации по подготовке школьников.

## Материалы для публикации

После подключения админки новые документы можно будет добавлять через раздел **Материалы** с направлением `ВСОШ` и категорией `Школьный этап`.
